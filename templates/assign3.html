Assignment 3

String formatting and operations. Yuy.

The old adage goes,
"He who can tame a wild woman is impressive.
Yet he who can index and substring by memory is feared."

Okay, that's not really an adage, but it should be.
String operations are some of the most important in any programming language. Being able to manipulate strings is required in almost all programs. Luckily, Objective-C makes string operations relatively simple with all of its different methods. Knowing the methods exist is the hardest part.

First off, we already know that a string must be set using a pointer and be assigned to an Objective-C string (eg NSString *fat = @"fat apples"). Now time for some new stuff.
Objective-C contains 2 types of string classes: NSString, and NSMutableString. NSString is immutable, ie it cannot be changed after initialization. Such is obviously not true of its counterpart. NSMutableString is a subclass of NSString, meaning all NSString methods will work on NSMutableString. However, not all of NSMutableString methods will work on NSString.
There are many different string operations. Examples include enumeration, comparison, concatenation, searching, subdividing, replacing, tokenizing, etc.
We will attempt to learn a good number of these.

The actual assignment

Part 1

Create 4 Objective-C strings (henceforth referred to as just strings) containing "miv", "reverof", "si", and "efil" and print out the strings in reverse.

Part 2

Create an array containing the strings "middleSchoolJoe.mp4", "babyjoe.mp4", and "babyjoe.mp4" (duplicated on purpose, because, I mean, who can have enough). Then, write a function that enumerates the array and prints out elements that are the same (index and value). Then, write a function that prints out which elements have the same suffix.

Part 3

Create two strings "Fancy" and "Feast". Concatenate them into one string and print out the new string.

Part 4

Create a string containing "He was number one". Then, create a function that prints out that checks the string for the word "number" and prints out the location of the word. Then, create a new string by removing the word number from the existing string (do not just create a new string with "he was one").

Part 5

Create a string "LEEEEEROYYYYYYYY JENKINSSSSSSS". Then, create a function that finds the letter J in the string and returns a substring from that letter until the end of the string.

Part 6

Create a mutable string "Happy". Append the word "Feet". You're welcome.

Part 7

Change all of the letters in the string "lEeEtHaXxOoRrZz" to uppercase and print out the string.
